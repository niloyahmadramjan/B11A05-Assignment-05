What are the different ways to select an element in the DOM?

Here are the different ways to select elements in the DOM

ID select : document.getElementById("myId") => Return a single elemen

Class Name select : document.getElementsByClassName("myClass") => it will Returns an HTMLCollection

Tag Name select : document.getElementsByTagName("p") => it will Returns an HTMLCollection

CSS selector first match : document.querySelector(".myClass") => it will return first match.

CSS selector all match : document.querySelector(".myClass") => it will return NodeList.

What is the difference between innerHTML, innerText, and textContent ?

Here is the difference between innerHTML, innerText, and textContent.

innerHTML - Returns or sets HTML content, parses HTML tags

innerText - Returns or sets only visible text, ignores hidden text, does not parse HTML.

textContent - Returns or sets all text (including hidden), does not parse HTML, fastest method.

What is event delegation in the DOM?

Event Delegation is a technique in the DOM where a parent element handles events for its child elements,
even if they are added dynamically.

It working instead of adding event listeners to multiple child elements, you attach a single event listener to their parent
and use event bubbling to detect the target.

What is event bubbling in the DOM?

Event Bubbling is a DOM event propagation mechanism where an event starts from the target element and moves
upward through its parent elements until it reaches the root (document).
